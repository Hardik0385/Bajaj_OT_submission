# Bajaj Finserv Health Limited — Full Stack Engineering Challenge

This repository contains the complete full-stack solution for the **SRM - BFHL Full Stack Engineering Challenge**. The application provides a robust backend API capable of parsing complex directed graph edges and assembling node hierarchies, along with a sleek, dynamically-rendered frontend interface.

## 🚀 Live Application
- **Production URL**: [https://hardik-agrawal-submission.onrender.com](https://hardik-agrawal-submission.onrender.com)
  *(This monolithic URL serves both the static frontend UI and the backend API.)*

## 🛠 Features & Capabilities

### Backend Strategy (`server.js`)
- **Advanced Graph Resolution**: Calculates accurate nested depths and builds recursive hierarchical objects out of randomized raw input strings (e.g., `A->B`).
- **Comprehensive Edge Case Protections**:
  - Validates and escapes non-string data types, nulls, and weird spacings.
  - Mitigates multi-parent structures by gracefully adopting the first logical parent and discarding conflicting edges.
  - Employs deep-stack **Cycle Detection** leveraging standard state-mapping DFS logic to detect loops (e.g., `A->B`, `B->C`, `C->A`) and returns empty cycle arrays to prevent memory stack overflows.
- **RESTful Endpoints**:
  - `POST /bfhl`: Ingests and solves the array of directional paths.
  - `GET /bfhl`: System health check endpoint.

### Frontend Strategy (`public/`)
- **Neumorphic Institutional UI**: Designed meticulously with a clean, high-contrast flat layout referencing official Bajaj Finserv brand alignments (Bajaj Blue `#005EB8`).
- **Dynamic Render Visualizations**: Capable of reading complex `JSON` responses to generate live visual node trees or cyclic warning cards.
- **Data Integrity Overlays**: Real-time modal inspections for viewing the raw JSON hierarchy structures generated by the Express.js Backend.

## 💻 Tech Stack
- **Backend Environment**: Node.js, Express.js, CORS
- **Frontend Architecture**: Vanilla HTML5, Structured CSS3 Custom Properites, Vanilla JavaScript
- **Test Suite**: Node execution (`test.js` covering 97 targeted input stress tests)

## 🔧 Running Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Hardik0385/Bajaj_OT_submission.git
   cd Bajaj_OT_submission
   ```

2. **Install core dependencies:**
   ```bash
   npm install
   ```

3. **Spin up the server:**
   ```bash
   node server.js
   ```

4. **View the Application:**
   Open your browser and navigate to `http://localhost:3000`.
